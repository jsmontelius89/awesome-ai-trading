# Contributing to Awesome AI Trading

Thanks for your interest in contributing! 🎉

## Guidelines

- **One resource per line** — keep descriptions concise.
- **Add to the correct section** — if unsure, mention it in the PR.
- **No duplicates** — check if the resource already exists.
- **Quality over quantity** — only add genuinely useful resources.
- **Format:** `- [Name](URL) — Short description.`

## How to Contribute

1. Fork this repository
2. Create a branch: `git checkout -b add-resource`
3. Add your resource
4. Commit: `git commit -m "Add [resource name]"`
5. Push and open a Pull Request

## What We're Looking For

- Open-source libraries and tools
- High-quality tutorials and courses
- Important research papers
- Active communities and blogs
- Free or freemium data sources

Thank you! 🚀
